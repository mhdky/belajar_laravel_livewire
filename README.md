## Cara Menggunakan Laravel Livewire
1. Install project laravel
2. Install Livewire
``` 
composer required livewire/livewire 
```

3. Jalankan local server
``` 
php artisan serve 
```
4. Tambahkan pada head ` @livewireStyles `
5. Tambahkan di atas tutup body ` @livewireScripts `
